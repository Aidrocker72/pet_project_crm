# Vue Mini CRM

Vue Mini CRM - это мини-приложение CRM (Customer Relationship Management) с открытым исходным кодом, разработанное с использованием современных технологий Vue.js, TypeScript и Vite. Этот проект демонстрирует лучшие практики разработки на Vue 3 с Composition API и модульной архитектурой.

## Особенности

- **Управление клиентами**: Создание, чтение, обновление и удаление записей клиентов
- **Управление сделками**: Отслеживание сделок с возможностью указания вероятности и стоимости
- **Воронка продаж**: Визуализация стадий продаж с возможностью перетаскивания сделок между стадиями
- **Панель управления**: Обзор ключевых метрик и недавней активности
- **Адаптивный дизайн**: Работает на различных размерах экранов
- **Полная типизация**: Через TypeScript
- **Модульная архитектура**: Четкое разделение на слои (entities, features, pages, shared)

## Технологии

- **Vue 3** - прогрессивный фреймворк JavaScript
- **TypeScript** - надстройка над JavaScript с проверкой типов
- **Vite** - быстрая среда сборки
- **Vue Router** - маршрутизация
- **Pinia** - управление состоянием
- **Vue Test Utils** - тестирование
- **Sass** - препроцессор CSS

## Архитектура

Проект следует архитектуре Feature-Sliced Design (FSD):

```
src/
├── entities/                 # Бизнес-сущности (Customer, Deal, Pipeline)
│   ├── customer/             # Сущность клиента
│   │   ├── model/            # Модель сущности
│   │   ├── api/              # Слой API
│   │   └── store/            # Хранилище Pinia
│   ├── deal/                 # Сущность сделки
│   └── pipeline/             # Сущность воронки
├── features/                 # Функциональные возможности
│   ├── customer-form/        # Форма клиента
│   ├── deal-form/            # Форма сделки
│   └── pipeline-board/       # Доска воронки
├── pages/                    # Страницы приложения
├── shared/                   # Общие компоненты и утилиты
│   ├── ui/                   # Переиспользуемые компоненты (Input, Button)
│   ├── types/                # Общие типы TypeScript
│   ├── styles/               # Общие стили, переменные и миксины
│   └── lib/                  # Вспомогательные библиотеки
└── widgets/                  # Компоненты верхнего уровня (Dashboard)
```

## Установка и запуск

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/yourusername/vue-mini-crm.git
   cd vue-mini-crm
   ```

2. **Установите зависимости**:
   ```bash
   npm install
   ```

3. **Запустите приложение в режиме разработки**:
   ```bash
   npm run dev
   ```
   Приложение будет доступно по адресу [http://localhost:5173](http://localhost:5173)

4. **Для сборки проекта в продакшен**:
   ```bash
   npm run build
   ```

5. **Для проверки сборки локально**:
   ```bash
   npm run preview
   ```

## Скрипты

- `npm run dev` - запуск сервера разработки
- `npm run build` - сборка проекта для продакшена
- `npm run preview` - предварительный просмотр продакшен-сборки
- `npm run test:unit` - запуск юнит-тестов
- `npm run type-check` - проверка типов
- `npm run lint` - проверка и исправление кода по стандартам

## Структура проекта

- `entities/` - бизнес-сущности приложения с полной изоляцией
- `features/` - функциональные возможности, связывающие сущности
- `pages/` - страницы приложения, объединяющие функциональности
- `shared/` - переиспользуемые компоненты, типы и утилиты
- `widgets/` - комплексные компоненты, содержащие бизнес-логику

## Локализация

Приложение полностью переведено на русский язык. Все тексты интерфейса, сообщения об ошибках, заголовки и названия кнопок отображаются на русском языке.

## Тестирование

Проект использует Vitest и Vue Test Utils для юнит-тестирования компонентов и бизнес-логики. Тесты находятся в папках `__tests__` внутри соответствующих модулей.
